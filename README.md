# MiniCashier（入力式レジ）

## できること
- 商品金額と支払額を入力して、お釣り or 不足金額を表示します
- 不正な入力（数字以外 / マイナス等）をチェックします

## 実行方法（IntelliJ）
1. `src/MiniCashier.java` を開く
2. ▶︎（Run）で実行
3. コンソールに数字を入力して Enter

## 入力例
- 商品金額: 680
- 支払額: 1000

## 出力例
- 商品金額: 680円
- 支払額: 1000円
- お釣り: 320円
